3. Procédure d'installation locale (XAMPP)
Pour tester ce site sur un serveur local XAMPP, suivez ces étapes :

1. Lancez le panneau de contrôle XAMPP.
2. Démarrez le module **Apache** (bouton "Start").
3. Ouvrez le dossier d'installation de XAMPP (généralement `C:\xampp` sur Windows ou `/Applications/XAMPP` sur Mac).
4. Allez dans le dossier `htdocs`.
5. Créez un nouveau dossier nommé `sae105`.
6. Copiez tous les fichiers de cette archive (index.html, style.css, scripts, dossier assets) à l'intérieur de ce dossier `htdocs/sae105`.
7. Ouvrez votre navigateur web.
8. Accédez à l'URL : `http://localhost/sae105`.
